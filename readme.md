# Netflix Data Analysis (EDA Project)

## Project Overview
- This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover insights about content distribution, trends, and viewer preferences.
- The analysis focuses on understanding how Netflix’s content library has evolved over time.
- The objective is to extract meaningful insights that can support content strategy and decision-making.

---

## Problem Statement
- With the rapid growth of streaming platforms, understanding content trends is essential.
- This project analyzes Netflix data to identify patterns in content type, genre, and release trends.

### Key Questions:
- What type of content is more prevalent: Movies or TV Shows?
- How has Netflix content grown over the years?
- Which countries produce the most content?
- What are the most popular genres on Netflix?

---

## Dataset Description
- The dataset contains:
  - Title
  - Type (Movie/TV Show)
  - Director
  - Cast
  - Country
  - Release Year
  - Rating
  - Duration
  - Genre (Listed In)

### Data Characteristics:
- Combination of categorical and numerical variables
- Contains missing values and inconsistencies

---

## Objectives
- Perform data cleaning and preprocessing
- Conduct detailed EDA
- Identify trends in content distribution
- Analyze genre and country-wise patterns
- Generate insights for content strategy

## Exploratory Data Analysis (EDA)

### Univariate Analysis
- Distribution of content type, ratings, and release years

### Bivariate Analysis
- Content type vs release trends
- Country vs number of titles
- Genre vs popularity

### Multivariate Analysis
- Combined analysis of country, genre, and release year

---

## Key Insights
- Movies dominate the platform compared to TV Shows
- Rapid content growth observed after 2015
- Few countries contribute majority of content
- Drama and Comedy are among the most common genres
